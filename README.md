Aura Coffee - "No One Blends Like Us"

A professional web application for a premium coffee shop, designed to provide a seamless digital experience for coffee lovers. This project features a warm, coffee-inspired aesthetic and a multi-page structure to showcase products and services.

Features
Home Page: A welcoming hero section with an overview of the brand's mission and "About Us" information.
Menu & Pricing: A categorized digital menu featuring detailed listings for Hot and Cold Coffees with pricing in LKR.
Special Offers: A dedicated page for promotional deals and seasonal specialties.
User Account System: Includes a functional login and registration interface for customer accounts.
Team Showcase: Displays the "Master Chefs" and baristas behind the coffee creations.
Contact Information: Direct lines of communication including a contact form, phone support, and physical location details.

Technologies Used
Frontend: HTML5, CSS3, JavaScript.

Frameworks: Bootstrap 5.0 for responsive design.

Fonts & Icons: Google Web Fonts (Open Sans, Oswald, Pacifico, Dancing Script) and Font Awesome/Bootstrap Icons.

File Structure
index.html: Main landing page.
menu.html: Detailed product listings and prices.
offers.html: Special deals and discounts.
team.html: Information about the team.
login.html: Secure access for users.
contact.html: Communication and inquiry page.
css/: Directory containing Bootstrap and custom stylesheets.
img/: Assets including logos, product images, and background graphics.
